## **Brillendar**

### React Native Schedule SNS Application.

[안드로이드 APK](https://github.com/ShinSeungWoo8987/Brillendar/raw/main/Brillendar.apk)

---

### Default

- TypeScript

<br/>

### Front End

- React Native

<br/>

### Back End

- Express

- GraphQL

- Redis

<br/>

### Database

- MySQL

- MongoDB

<br/>

### Cloud

- AWS EC2

- AWS S3

- AWS RDS

- NCP SMS

---

### 주요기능

- Auth

  - 로그인
  - 회원가입
    - 이메일 인증
    - 핸드폰 인증 (NCP SMS)

- Follow

  - 팔로우 / 언팔로우
  - 팔로워 / 팔로잉 목록
  - 비공개 계정 설정
  - 팔로우 요청
  - 팔로우 수락

- Schedule
  - 일정등록
  - 공개범위 설정 (나만보기)
  - 결과등록 (이미지 업로드
  - 좋아요 / 좋아요 목록
  - 댓글
- Feed
  - 현재 진행중인 스케줄
  - 오늘하루 스케줄 요약
